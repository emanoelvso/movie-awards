## Description

The Movie Awards API.

## Project setup

The Node version used: 22.19.0

```bash
$ yarn install
```

## Database

```bash
# Generate Migrations
yarn run db:generate

# Migrate Migrations
yarn run db:migrate

# Drop Migrations
yarn run db:drop
```

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev
```

## Run tests

```bash
# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```

## Run Docs

Access [API Documentation](http:localhost:3000/docs)
